# Random-Task-Generator-

Генератор случайных задач
Автор: Горбунов Даниил

Описание
Данная программа — это простое и удобное настольное приложение на Python для генерации случайных задач из вашего списка, их хранения и истории выполнения. Вы можете добавлять, удалять и генерировать задачи, а также видеть историю с возможностью фильтрации (по типу задания).

Пример использования
Добавьте задачу:
Введите задание и его тип (учёба, спорт, работа) слева и нажмите "Добавить".

Удалите лишнюю:
Выберите задачу в списке и нажмите "Удалить задачу".

Получите случайную задачу:
В центральной части нажмите на "Сгенерировать задачу". Программа покажет задание и добавит его в историю.

История:
В правой части отображается история ваших сгенерированных задач. Можно фильтровать по типу или очистить историю.-
